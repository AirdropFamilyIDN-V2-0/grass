# How to get userid grass
- login ke web grass , terus inspect / f12 ,  klik console
- paste
``` 
localStorage.getItem('userId')
```
- kalau ga bisa paste ketik ```allow pasting``` enter
- baru paste ```localStorage.getItem('userId')```enter lagi
